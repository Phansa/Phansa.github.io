IFMP Database Application

I was asked to create this application in the summer of 2015 in order to help this nonprofit organization keep track of its members. I wrote this application in VB.Net and it access a MSSQL database. This project taught me a lot about meeting specifications and how to create a UI that would be intuitive and simple to use for the user. Up until this project, I was mostly coding things that used the command line only so creating a GUI was a good learning experience. I will outline some of the features of this program here.

This is the basic login Screen. You can log onto this application as either a user or and administrator.

This is the screen that allows you to log into the database. You will need to have your IP added to a list of IP's allowed to access the database before you can use this application however. If your ip isn't on that list then you cannot proceed further.

This is the main user interface. It contains many features that I will go over. For demonstration purposes I have excluded all actual members and am just using fake members I created named test one and test two. A feature of this application is the ability to search for members by either typing part of their first/last name or typing their ID into the corresponding textboxes.

Last Name Search

A user may click on the ID column of any member and autofill the textboxes with the data stored in the database corresponding to that member.

A user may fill in textboxes then press update in order to update the information stored in the database.
