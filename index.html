Computer Science Projects

I have worked on an open source project called Pokemon Snowdown. This game is a Pokemon battle simulator that allows players to test their Pokemon battling sklls against the CPU. I have created the whole wiki for this game, which can be found here .

I have also created a database application in Visual Basic to keep track of membership information for India Foundation of Metropolitan Princeton. More details about both of those projects can be found here .

I have worked on another open source Pokemon game simulator, FullScreen Pokemon which is a TypeScript implementation of the original Pokemon games with an emphasis on extensibility and modularity. I worked on various parts of the game engine, fixed some bugs and even animated some moves! It is a very cool and ambitious project and I would encourage you to check it out or even contribute to it!
